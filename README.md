# PicsumPhotos
This application is useful to retrive list of photos. I have used pblic API for network call and fetched photo details.

Light | Dark
:-: | :-:

## Requirements:
### Functional:
- Fetching and parsing data from backend
- Custom element controls.
- List photos.
- Pagination.
- Photo details.
- Dark mode.
- Dynamic fonts
- Accessibility
- Localisation

### Non-Functional:
- Scalability:
  - Modularity.
  
- Reliability:
  - Unit tests.

## Design Decisions
- UI:
  - UIKit
  - MVVM Architecture
  - Repository pattern
  - Coordinator pattern
  - Adaptor pattern

- Principals and Patterns:
  - SOLID conformance:
    - Features are separated into modules.
    - UseCases for business logic.
  - Dependency injection.

- Dependency Manager:
  - Swift Package Manager
